# Story Command

Create a user story with structure, acceptance criteria, and estimates.

## Instructions

When this command is invoked, load the **project-management** skill for:
- Story templates (feature, bug, technical, spike)
- Acceptance criteria guidelines (Given/When/Then format)
- Estimation guide (T-shirt sizing â†’ story points)
- Story structure and best practices

Then load the **agile-board** skill to:
- Create the story on the configured board (ZenHub/Jira/Linear)
- Link to epic/parent
- Add labels and assign to sprint

The skills contain complete story templates and board integration.
