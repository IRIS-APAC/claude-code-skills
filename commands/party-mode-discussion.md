# Party Mode Command

Bring multiple IRIS agent personas into one session for collaborative multi-agent discussion.

## Instructions

When this command is invoked:

1. **Load the party-mode skill** for:
   - Agent manifest loading and roster building
   - Multi-agent conversation orchestration
   - Character-consistent role-playing across diverse expert personas
   - Discussion facilitation and moderation

2. **Party mode flow**:
   - Load all available IRIS agent personas from the agent manifest
   - Welcome the user and introduce the collaborating agents
   - Ask what topic or question the user wants to discuss
   - Orchestrate natural multi-agent conversations with 2-3 relevant agents per round
   - Rotate agent participation for diverse perspectives

3. **Agent intelligence**:
   - Automatically select the most relevant agents for each topic
   - Allow natural cross-talk and agent-to-agent interactions
   - Maintain each agent's unique communication style and expertise
   - Enable agents to build on and respectfully disagree with each other

4. **Exit party mode**:
   - Type `*exit`, `goodbye`, `end party`, or `quit` to leave
   - The facilitator will summarize key points before closing

**Purpose**: Get diverse expert perspectives on any topic through engaging multi-persona collaboration.

The party-mode skill contains complete agent orchestration, personality management, and conversation flow guidance.
