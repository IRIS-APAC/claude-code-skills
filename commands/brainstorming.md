# Brainstorming Command

Facilitate an interactive brainstorming session using diverse creative techniques and structured ideation methods.

## Instructions

When this command is invoked:

1. **Load the brainstorming skill** for:
   - Session setup and topic discovery
   - Creative technique selection from the brain-methods library
   - Anti-bias protocols to push past obvious ideas
   - Structured ideation with a goal of 100+ ideas before organizing
   - Session documentation and synthesis

2. **Session flow**:
   - Discover the brainstorming topic and context
   - Select and apply creative techniques iteratively
   - Keep the user in generative exploration mode
   - Offer **advanced-elicitation** for deeper exploration of promising threads
   - Synthesize findings into a brainstorming report

3. **Integration with other workflows**:
   - After session, offer to feed ideas into `/iris-bmm-research` for validation
   - Or into `/requirements-design` for formal requirements gathering

**Output**: `brainstorming-session-{date}.md` with all ideas, techniques used, and synthesis.

The brainstorming skill contains complete session management, technique library, and facilitation guidance.
